# Voice Message Language Identifier

Этот проект представляет собой Telegram-бота, который определяет язык голосового сообщения (Русский, Английский, Французский, Немецкий, Испанский) с использованием современных методов машинного обучения.

## Авторы
*   **Малышева Анастасия**
*   **Бурмистрова Юлия**

## Цель проекта
Разработать и сравнить различные подходы к задаче идентификации языка по аудиозаписи и внедрить лучшую модель в удобный интерфейс Telegram-бота.

## Данные
Для обучения использовался датасет **VoxLingua107**. Это обширный набор данных, содержащий речевые сегменты на различных языках, извлеченные из видео YouTube.
*   **Классы:** Русский, Английский, Французский, Немецкий, Испанский.

## Методы и Подходы

В рамках исследования были протестированы два основных подхода:

### 1. Классический ML (Baseline)
*   **Признаки:** Спектрограммы (Mel-spectrograms).
*   **Модели:** Логистическая регрессия, Random Forest, SVM.
*   **Результат:** Базовое качество, чувствительность к шумам и длине записи.
*   *Код:* `classic-ml-2.ipynb`

### 2. Deep Learning (HuBERT Embeddings)
*   **Признаки:** Эмбеддинги, извлеченные из предобученной модели **HuBERT** (Hidden Unit BERT).
*   **Преимущества:** Модель учитывает контекст и семантические особенности речи, что позволяет получать векторы высокого качества даже для коротких аудио.
*   **Результат:** Значительный прирост точности по сравнению с классическими методами.
*   *Код:* `hubert-emb.ipynb`

## Telegram Бот
Лучшая модель (на основе HuBERT) была интегрирована в Telegram-бота.
*   **Функционал:** Пользователь отправляет голосовое сообщение -> Бот определяет язык.

## Результаты
Использование предобученных эмбеддингов HuBERT показало наилучшие результаты метрик (Accuracy/F1) по сравнению с обучением классических классификаторов на "сырых" спектрограммах. Подробные графики и Confusion Matrix доступны в презентации и ноутбуках.
